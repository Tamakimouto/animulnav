Overview
--------
A simple jQuery plugin to animate all the elements of a "ul".

Simple Usage
-----
Import the js file 
	
	<script src="animulnav-0.1.min.js" type="text/javascript"></script>

Add the function to your ul element:
	
	$('ul.class').animulnav();
